# Notion 課金が必要なシチュエーション

## 💰 料金体系

### Notion プラン（個人向け）

| プラン | 価格 | 制限 |
|--------|------|------|
| **Free（無料）** | $0/月 | ✅ ほぼ無制限（制限少ない） |
| **Plus** | $12/月 | 個人・少人数向け |
| **Business** | $27/月 | チーム向け |
| **Enterprise** | 要見積 | 大企業向け |

---

## ✅ Free プラン（無料）で十分な場合

**Karpathy Wiki 構築には Free で大丈夫** ❌ 課金不要

```
✅ できること（Free プランで）:
  • ページ数: 無制限
  • Database: 無制限
  • ファイルアップロード: 無制限（5MB/ファイル）
  • Block 数: 無制限
  • API 呼び出し: 月間リクエスト数に制限あり
  
✅ 知識管理には Free で十分
✅ 個人で使う分には Free で十分
✅ AI自動要約（Claude API）: 別途課金（下記参照）
```

---

## 🚨 課金が必要になるシチュエーション

### シチュエーション 1: **チームで使う**（5人以上）

```
Free プラン: 1人
  └─ 個人のみ使用可能

👥 複数人で使う場合:
  ├─ Plus ($12/月): 最大5人まで追加可能
  ├─ Business ($27/月): 無制限の人数
  └─ Enterprise: 要見積
```

**あなたの場合:** 個人利用なら Free のまま ✅

---

### シチュエーション 2: **API で大量アクセス**

Notion API の呼び出し制限：

```
Free プラン:
  ├─ 月間リクエスト: 制限あり（明確な数字は非公開）
  ├─ 実際には: 数千リクエスト/月 は可能
  └─ Claude 自動投稿: 1記事 = 数リクエスト
  
→ 月に 100 記事程度なら Free で OK
→ 月に 1000+ 記事なら Plus/Business 推奨
```

**あなたの場合:** 
- 週に 5-10 記事程度なら Free で十分
- 高頻度投稿（毎日複数）なら Plus 検討

---

### シチュエーション 3: **高度なワークスペース機能**

**Free では制限される機能:**

```
✅ Free でできる:
  • Database
  • Pages
  • リレーション
  • フィルター・ソート
  • シンプルな UI

❌ Plus/Business で必要:
  • ワークスペースメンバー管理（複数人）
  • 高度なアクセス権限制御
  • ワークスペース内の共有設定
  • Advanced API機能
```

**あなたの場合:** 個人利用なら不要 ✅

---

### シチュエーション 4: **Claude API（重要！）**

Notion の課金ではなく **Claude API の課金**:

```
Notion 無料:
  └─ Notion そのものは無料

+

Claude API 有料:
  └─ 記事をClaudeで自動要約: 有料
  
例：
  • 月100記事 × 1000トークン/記事 = 100,000トークン
  • 100,000トークン × $0.003/1000 = ~$0.30/月
  
→ 非常に安い（月数円～数百円程度）
```

**あなたの場合:** Claude API は既に設定済み ✅
